Первый запуск

При первом запуске приложение создает внутреннюю библиотеку модулей и добавляет несколько предустановленных модулей: Синодальный перевод, UBIO и King James Version.

Для хранения пользовательских файлов используется внешняя папка BibleAxis на устройстве. В ней хранятся все файлы, необходимые для работы приложения. Основная папка внутри неё — modules, где размещаются модули с текстами Библии и другими книгами, которые можно читать в приложении.

Для первоначального создания папки BibleAxis перейдите в библиотеку и нажмите «Выбрать место хранения…». Укажите папку, где приложение должно создать структуру BibleAxis/modules. Эта настройка выполняется один раз.

Программа поддерживает модули, совместимые с приложением «Цитаты из Библии» (Bible Quote): https://www.biblequote.org/.

Вы можете добавлять собственные модули. В интернете существует множество сайтов, где их можно скачать. Для поиска используйте запрос: «Модули для Цитаты из Библии (Bible Quote)».

После скачивания поместите ZIP-файлы модулей в папку BibleAxis/modules на вашем устройстве.

Расположение файлов внутри BibleAxis/modules может быть любым. Вы можете организовать папки так, как вам удобно. Например:

  • отдельная папка для переводов Библии;
  • отдельная папка для книг;
  • внутри неё — папки для богословских, художественных и других категорий.

Глубина вложенности папок ограничена только возможностями системы Android.

Чтобы добавить новые или обновленные модули в приложение, откройте библиотеку и нажмите «Обновить». Эта кнопка копирует новые и измененные ZIP-модули во внутреннюю библиотеку приложения и обновляет список доступных модулей. Уже добавленные модули при этом не удаляются.

Если удалить приложение, папка BibleAxis/modules останется на устройстве. После повторной установки достаточно снова выбрать то же место хранения и нажать «Обновить».

Окно чтения (общее)

Выбор модуля и книги. После первого запуска открывается окно выбора модуля. Затем выбирается книга, а после — глава (если книга имеет деление на главы).

При следующем запуске приложение открывает последний прочитанный текст.

Чтобы выбрать другой модуль, книгу или главу, откройте Библиотеку из бокового меню. Окно выбора также можно открыть, нажав на название текущего модуля и книги в верхней панели.

Окно чтения. В приложении есть два режима: простой и режим чтения. Переключение между ними выполняется двойным нажатием по экрану.

В верхней панели доступны быстрые действия:

  • поиск по тексту;
  • история.

Дополнительные функции (Библиотека, Закладки, Настройки, Справка, О программе, переключатели режима и другие) находятся в боковом меню.

Простой режим. Текст прокручивается обычной вертикальной прокруткой. Если соответствующая настройка включена, для прокрутки можно использовать кнопки громкости.

Режим чтения. Навигация выполняется с помощью тап-зон экрана:

  • верх/низ — прокрутка текущего текста;
  • левая часть — переход к предыдущей главе;
  • правая часть — переход к следующей главе.

Долгое нажатие на экран открывает окно выбора главы.

Режим день/ночь. Для чтения в темноте можно включить ночной режим. Он переключается из бокового меню и изменяет цветовую схему текста и фона.

Настройка шрифта и фона. В настройках можно изменить:

  • цвет текста и фона;
  • размер шрифта;
  • межстрочный интервал;
  • гарнитуру шрифта.

Автоповорот экрана. В настройках можно отключить автоматический поворот экрана, чтобы зафиксировать ориентацию во время чтения.

Поиск по текущей странице. Приложение поддерживает поиск по текущей странице. Эта функция включается в настройках, после чего кнопка поиска становится доступной в верхней панели окна чтения.

Работа со стихами (с текстом)

Работа со стихами выполняется в простом режиме. Чтобы выбрать стих, нажмите по нему. Для выбора нескольких стихов нажмите по нужным стихам последовательно.

После выбора открывается меню действий: Закладки, Поделиться, Ссылки и Копирование.

Закладки. Нажмите кнопку Закладки, чтобы добавить выбранный стих или отрывок в закладки. Доступ к закладкам открывается из бокового меню. В окне закладок можно просматривать, сортировать и удалять записи. Для удаления одной закладки нажмите и удерживайте её, затем подтвердите удаление.

Поделиться. Функция Поделиться отправляет выбранный текст через доступные приложения (мессенджеры, почту и другие).

Ссылки. Функция Ссылки открывает список параллельных мест для выбранного стиха. При нажатии на ссылку открывается соответствующий текст. В настройках можно включить отображение контекста в списке ссылок.

Копирование. Кнопка Копирование копирует выбранный текст в буфер обмена, после чего его можно вставить в любое приложение.

Для функций Поделиться и Копирование есть дополнительные настройки. С их помощью можно настроить разбивку текста по стихам, добавление ссылки к отрывку, положение ссылки, короткую или длинную форму ссылки, а также добавление имени модуля.

Цветовая разметка текста. В приложении можно выделять цветом слова, один стих или несколько стихов. После выделения текста выберите нужный цвет в палитре подсветки. Показ цветовой разметки можно включать и отключать в боковом меню для удобства чтения.

История (функция)

История помогает быстро возвращаться к ранее открытым местам Писания и книг. Открыть Историю можно из верхней панели окна чтения (иконка часов).

В окне Истории отображается список последних открытых мест в формате: модуль, книга и глава. Нажатие на запись открывает соответствующий текст.

Размер истории можно настроить в настройках приложения. Доступные значения: от 10 до 100 записей.

Поиск по тексту

Поиск открывается из окна чтения нажатием на кнопку поиска в верхней панели.

Поиск выполняется по библейским модулям. Можно выбрать один или несколько модулей для одновременного поиска.

Запрос. Введите слово или фразу в поле поиска и нажмите кнопку Поиск (или кнопку поиска на клавиатуре).

Выбор модулей. В поле Модули можно указать, где выполнять поиск:

  • один модуль;
  • несколько модулей одновременно.

Если не выбран ни один модуль, запуск поиска невозможен.

Область поиска. Доступны следующие варианты:

  • Вся Библия;
  • Ветхий Завет;
  • Новый Завет;
  • Одна книга;
  • Выбранные книги;
  • Диапазон книг (от книги до книги).

Для режимов Одна книга, Выбранные книги и Диапазон книг нужно дополнительно указать соответствующие книги.

Дополнительный параметр. Опция Искать только целые слова позволяет исключить частичные совпадения внутри других слов.

Результаты поиска. После завершения поиска отображается список найденных мест. Нажатие на результат открывает соответствующий текст в окне чтения. При возврате в поиск можно продолжить работу с текущими результатами.

Новый поиск. Для нового поиска откройте панель настроек поиска и при необходимости измените запрос, модули, область поиска и дополнительные параметры.

Важно. Если в приложении нет доступных библейских модулей, поиск по тексту будет недоступен до их добавления.
